## 🎮 How to Add BaseLib-StS2 to Slay the Spire 2

1. Locate your Slay the Spire 2 game folder. This is often found in:
   - `C:\Program Files (x86)\Steam\steamapps\common\Slay the Spire 2`
   Or your chosen installation directory.
2. In the game folder, find the "mods" folder. If there is none, create a new folder called `mods`.
3. Copy the extracted BaseLib-StS2 folder into the "mods" folder.
4. Launch Slay the Spire 2.
5. When the game menu appears, look for the "Mods" option.
6. Enable BaseLib-StS2 from the mod list.
7. Start or load your game normally. The mod is now active.

## 🔧 Troubleshooting Common Issues

**Mod Fails to Activate**  
- Verify you copied the correct folder into the mods directory. It should contain files, not just another folder inside.

**Game Crashes on Startup**  
- Remove the mod and restart the game to confirm if BaseLib-StS2 causes the issue.
- Ensure your game is fully updated.

**Mod Not Appearing in Mods List**  
- Check the mod folder name and structure.
- Make sure you are launching the right version of the game.
